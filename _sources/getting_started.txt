Getting Started
================

Introduction
-------------

Filers is a a project for acquiring and, manipulating video files,
as well as organizing and re-ordering files en-masse.

Usage
------

To use Filers, you first need to install it, see :ref:`install-filers`.

After it's installed, it is run through the main module. For example::

    python -m filers.main

When using a packaged version of filers, one starts it by 
just running filers.exe.

Complete API documentation is at :ref:`filers-root-api`.
